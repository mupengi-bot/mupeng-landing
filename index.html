자주 묻는 질문

현재 무펭이(OpenClaw)는 macOS에서 가장 안정적으로 동작합니다. Mac mini 하나만 있으면 24시간 서버처럼 활용 가능합니다. Windows 지원은 준비 중입니다.
네, AI 모델 사용에 따른 API 비용(월 2~5만원 수준)은 별도입니다. 사용량에 따라 달라지며, 비용 최적화까지 함께 관리해드립니다.
무펭이는 고객의 Mac에서 로컬로 동작합니다. 데이터가 외부 서버에 저장되지 않으며, 필요한 API 호출만 암호화된 채널로 이루어집니다.
물론입니다. Basic 플랜으로 세팅만 받고 직접 운영하셔도 됩니다. 운영 가이드와 교육을 제공합니다.
세팅 완료 전 100% 환불, 세팅 완료 후 7일 이내 50% 환불이 가능합니다. 월 운영비는 해지 시 다음 달부터 청구되지 않습니다.
